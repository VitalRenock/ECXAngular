Bienvenue sur DropNote!

DropNote, anciennement ECX est un projet permettant de réaliser facilement de la documentation pour les développeurs.
Ce projet à été réalisé dans le cadre d'une formation sur les technologies '.NET' et 'Angular', proposée par le centre de formation 'TechnofuturTIC' de Gosselies.
